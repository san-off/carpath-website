# 洗車予約自動化・インセンティブ計測 運用計画

## 1. 目的
洗車サービスの予約業務を効率化（自動化）しつつ、Webサイトを経由させることでGA4での計測を確実に行い、運営者のインセンティブ実績を正確に反映させる。

## 2. システム構成
* **予約システム**: Googleカレンダー「予約スケジュール」機能
* **計測ツール**: Google Analytics 4 (GA4)
* **導線集約点**: Carpath公式サイト 洗車ページ (`wash.html`)

## 3. 実装・運用フロー

### ① Webサイト（wash.html）の改修
洗車ページのCTA（コールトゥアクション）を、従来の「LINE相談」から「WEB予約」へシフトさせる。
* **ボタン実装**: Googleカレンダーの予約スケジュールURLをリンク先に設定。
* **イベント計測**: ボタンクリック時にGA4イベント `click_wash_reserve` を発火させる。
  * これにより、1クリック＝50円（カテゴリーC）の成果としてカウントされる。

### ② LINE公式アカウントの導線変更
LINE内で予約を完結させず、必ずWebサイトを一度経由させる「実績取りこぼし防止」設定。
* **リッチメニュー**: 「洗車予約」ボタンのリンク先を `wash.html` に設定。
* **自動応答**: 予約に関するキーワードに対し、「空き状況の確認・予約はこちらから：[wash.htmlのURL]」と返信するよう設定。

### ③ Googleカレンダー予約フォームの設定
Googleアカウントを持たない顧客でも予約可能な設定とし、以下の項目を必須とする。
* **入力項目**: 氏名、メールアドレス、電話番号、車のサイズ、車種名。
* **サンクスページ/メール**: 予約完了後の案内文にLINE友だち追加URLを記載し、リピーター化を促進する。

## 4. 運用のメリット
* **運営者**: すべての予約導線がWebサイトのボタンを経由するため、インセンティブ実績が100%可視化される。
* **店舗（会社）**: 有料予約システムへの課金を回避しつつ、24時間自動受付を実現。電話対応コストを削減できる。
* **顧客**: 電話の手間なく、リアルタイムな空き状況を確認して即座に予約が可能になる。

## 5. 注意事項
* Googleカレンダー側で「洗車担当者のシフト」や「他の作業予定」とカレンダーを同期させ、ダブルブッキングを防止する。
* Webサイトのボタンが正常に計測されているか、定期的にGA4のリアルタイムレポートで確認を行う。
