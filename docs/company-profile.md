* **特記事項**: スズキ副代理店

## 2. 事業内容と強み
パルマスは、米子市において販売・整備・本社の機能を一拠点に集約し、ワンストップかつ垂直統合型のサービスを提供している。

* **車両販売（新車・中古車）**
  * **低在庫・高回転戦略（アセットライト）**: 物理的な展示在庫を常時5台程度に厳選し、在庫維持コストや値落ちリスクを抑制。キャッシュフローの流動性を高めている。
  * **注文販売と新車販売**: 在庫にない車両でも、全国のオークションからの調達や、スズキをはじめとする全メーカーの新車販売に対応。
* **車検・整備・品質管理**
  * 自社認証工場を完備し、国家資格を持つ熟練の整備士が対応。
  * 中古車の納車前には、オイル等消耗品の原則交換、車検基準以上の独自基準での部品交換、シートの染み抜きを含む徹底したクリーニングを実施。
* **洗車・コーティング**
  * **KeePer取扱店**として、圧倒的な艶と水弾きを実現するコーティングを提供。
  * 機械洗車では届かない細部や車内清掃まで徹底的に仕上げる、極上の手洗い洗車を実施。
* **買取・廃車事業**
  * 「廃車買取カーパス」として、米子市内および近隣エリアへの無料出張査定を実施（出張費・キャンセル料無料）。
  * リサイクルパーツの確保によるコスト低減に加え、低年式車からの乗り換え需要の捕捉、資源循環による社会貢献という多面的な意義を持つ。
* **自動車保険代理店事業**
  * 三井住友海上の代理店として、保険の最適なプラン提案から、万が一の事故時の修理手配、レッカー、保険会社への連絡までを自社で一括対応し、顧客負担を軽減している。

## 3. 経営理念・行動指針
* **経営理念**: 「未来の山陰の為にバトンを繋ぐ」
  * 車両の買取や廃車回収を通じ、次に必要とする方への引き継ぎや資源循環を行い、過疎化地域における社会インフラとしての役割を担う。
* **5つの精神**（従業員の行動指針）
  1. **誠実さ**: 車両瑕疵も含めた透明性のある説明。
  2. **社交性**: 地域社会・顧客との開かれた対話。
  3. **自制心**: 利益追求だけでなく、自社品質を維持する規律。
  4. **パートナーシップ**: 長きにわたるカーライフの伴走者。
  5. **勤勉さ**: 自動車技術の進化に対する継続的な学習。
