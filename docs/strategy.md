# Project: Carpas Web Marketing & Career Strategy 2026

## 1. プロジェクト概要
鳥取県米子市の自動車販売店「カーパス」におけるWeb集客サイトの構築、および運営を通じた個人のキャリア戦略プロジェクト。
現在の「事務職（ライスワーク）」をWebマーケティングスキルによって高付加価値化し、労働収入のみで世帯生活費（月25万円）を安定的に賄う基盤を構築することを目的とする。

## 2. プロジェクト背景（コンテキスト）
- **運営者:** 松下 陽介 (39歳)
- **居住地:** 鳥取県米子市
- **家族構成:** - 妻（市役所勤務・会計年度任用職員）
    - 長男（小5相当・重症心身障害児1級・施設入所中）
    - 長女（小3・軽度発達障害・特別支援学級併用）
- **現状:** - 世帯労働収入は約20〜22万円。残りを公的扶助（児童手当・特児手当）で補い生活費25万円を捻出している状態。
    - 手当に依存せず、労働収入（ライスワーク）だけで25万円を突破し、余剰分を自己実現（ライフワーク）に充てる戦略をとる。

## 3. Webサイト戦略
既存のJimdoサイトから、AI（生成AI）を活用して自作したHTML/CSSサイトへ完全移行する。

### 技術スタック
- **構造:** HTML5 / CSS3 / JavaScript (生成AIによる構築)
- **管理:** GitHubによるバージョン管理
- **ドメイン:** carpath.net (Jimdoより移管予定)
- **解析:** Google Analytics 4 (GA4) によるイベント計測

### サイト構成
- `index.html`: 総合窓口
- `cars.html`: 在庫車両一覧
- `car-detail-[id].html`: 車両詳細（最重要ページ）
- `maintenance.html`: 車検・整備
- `wash.html`: 手洗い洗車
- `coating.html`: コーティング
- `purchase.html`: 買取・廃車
- `insurance.html`: 自動車保険

## 4. インセンティブ（報奨金）モデル
社長（営業畑出身）に対し、Web成果に基づいた以下の「成果報酬型」インセンティブを提案・導入する。集計コスト削減のため、CTA（クリック）を成果指標の主軸とする。

### ① PV段階報酬（認知度評価）
- 月間 1,000 PV 達成ごとに 5,000円 加算。

### ② CTA（クリック）単価設定
ボタンクリック（電話・LINE相談）を「デジタル来店」と定義し、カテゴリー別に単価を設定。
- **カテゴリーA（300円/回）:** 車両詳細、在庫検索、買取・廃車
- **カテゴリーB（150円/回）:** 車検・整備、コーティング、保険
- **カテゴリーC（50円/回）:** 手洗い洗車
- **カテゴリーD（100円/回）:** トップページ総合窓口

## 5. 運用マニュアル & コンテンツ方針
- **計測:** GA4でクリックイベントを自動発火させ、月末にレポート出力。
- **コンテンツ:** 「米子市」「4WD」「雪国対策」など、地域特性（ローカルSEO）を重視した情報発信。
- **信頼構築:** 運営者の顔が見える情報の集約、LINEによる相談ハードルの低減。
