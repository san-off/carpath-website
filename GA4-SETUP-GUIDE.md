# 📈 GA4（Googleアナリティクス4）設定・計測マニュアル
本ドキュメントは、CarpathWeb集客プロジェクトにおける「成果報酬（インセンティブ）算出」のためのGA4設定およびカスタムイベントの集計手順をまとめたものです。
## ステップ1：GA4プロパティの作成と測定IDの取得
すでにGoogleアナリティクスのアカウントがある場合は、プロパティの作成から行います。
 1. Googleアナリティクス にログインする。
 2. 左下の ⚙️**管理（歯車アイコン）** をクリックする。
 3. プロパティ列の **「プロパティの作成」** をクリックし、画面の指示に従って作成する（プロパティ名例：Carpath公式サイト）。
 4. データストリームの設定で「ウェブ」を選択し、サイトのURLとストリーム名を入力して **「ストリームを作成」** をクリックする。
 5. 測定ID（G- から始まる文字列。例：G-12345ABCDE）が表示されるので、コピーして控えておく。
## ステップ2：HTMLへのトラッキングコード埋め込み
すべてのHTMLファイルの <head> 内にあるダミーコードを、実際のコードに差し替えます。
 1. 各HTMLファイルの <head> タグ内にある以下のコメントアウト（``）を削除します。
 2. G-XXXXXXXXXX となっている箇所（計2箇所）を、ステップ1で取得した自分の測定IDに書き換えます。
```html
<script async src="[https://www.googletagmanager.com/gtag/js?id=G-12345ABCDE](https://www.googletagmanager.com/gtag/js?id=G-12345ABCDE)"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'G-12345ABCDE');
</script>

```
※完了したらGitHubへ Commit & Push して本番環境に反映させます。
## ステップ3：カスタムディメンションの登録（超重要⚠️）
本サイトでは、ボタンが押された際に「どのページの」「どのURLで」押されたかを分析するため、独自のパラメータ（event_category, event_label, page_type）をGA4に送信しています。これらをGA4のレポート画面で見るために、以下の登録作業が必須です。
 1. GA4管理画面の左下 ⚙️**管理** ＞ **「カスタム定義」** をクリックする。
 2. **「カスタムディメンションを作成」** をクリックし、以下の3つを1つずつ登録する。
| ディメンション名 (自由入力) | 範囲 | 説明 (自由入力) | イベント パラメータ (完全一致で入力) |
|---|---|---|---|
| イベントカテゴリ | イベント | CTAクリックの分類 | event_category |
| イベントラベル（URL） | イベント | クリックが発生したパス | event_label |
| ページタイプ | イベント | ページの種類（cars等） | page_type |
※登録後、データがレポートに反映されるまで最大24時間程度かかります。
## ステップ4：計測のテスト（リアルタイム確認）
設定が正しく完了しているかテストします。
 1. 自分のスマホやPCで、本番環境のサイトにアクセスする。
 2. いずれかのCTAボタン（LINEや電話など）をクリックする。
 3. GA4の左メニューから **「レポート」** ＞ **「リアルタイム」** を開く。
 4. 下部にある「イベント数」のカード内に、click_top_line や click_car_detail_tel などのイベント名が表示されていれば計測成功です！
## ステップ5：コンバージョン（成果）の設定
インセンティブ集計を分かりやすくするため、特定のイベントを「コンバージョン（CV）」としてマークします。
 1. GA4管理画面の左下 ⚙️**管理** ＞ **「イベント」** をクリックする。
 2. サイトで発生したイベントの一覧が表示されます（※データ反映に数日かかる場合があります）。
 3. インセンティブ対象となるイベント（例：click_car_detail_line, click_wash_tel など）の右側にある **「コンバージョンとしてマークを付ける」** のトグルスイッチをオン（青色）にします。
一覧にまだ表示されていない場合は、⚙️管理 ＞「コンバージョン」から「新しいコンバージョンイベント」をクリックし、イベント名を正確に手入力して保存することで事前登録も可能です。
## 📊 インセンティブ（報酬）の集計方法
月末の集計作業は、GA4の「探索」機能を使って行います。
 1. 左メニューの **「探索」** ＞ **「空白」** をクリック。
 2. **変数**パネルの設定：
   * ディメンションに「イベント名」と「ページタイプ（ステップ3で作ったもの）」をインポート。
   * 指標に「イベント数」をインポート。
 3. **タブの設定**パネルへの配置：
   * 「行」に「イベント名」をドラッグ＆ドロップ。
   * 「値」に「イベント数」をドラッグ＆ドロップ。
 4. これで、click_car_detail_line が 〇〇回、click_wash_tel が 〇〇回……と一覧化されます。
 5. あとはスプレッドシート等で、この回数とインセンティブ単価（Aランク:300円、Bランク:150円...）を掛け算すれば、今月の報酬額が算出できます！
