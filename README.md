# Carpath（カーパス）Webサイト 管理マニュアル

このリポジトリは、株式会社パルマスのホームページのソースコードを管理しています。

## 🚗 車両詳細ページの管理ルール

### 【命名規則】
新規追加時や、同じ車を再販する時の重複を防ぐため、以下のルールでファイルを作成します。
`car-detail-[車種]-[車台番号下4桁]-[仕入年月].html`
（例：`car-detail-tanto-5678-202605.html`）

---

### 1. 新規車両を追加する手順
1. `car-detail.html`（テンプレート）をコピーし、上記の命名規則で新しいファイルを作成する。
2. 新しいファイル内の「写真・価格・スペック・説明文」を書き換える。
3. `cars.html`（在庫一覧ページ）に新しい車両のカード（枠）を追加する。
4. 一覧ページの「車両詳細を見る」ボタンのリンクを、新しく作ったファイル名に設定する。
   `<a href="car-detail-tanto-5678-202605.html" class="...">`
5. GitHubでセーブ（Commit & Push）する。

---

### 2. 車両が売れた場合（SOLD OUT表示にする手順）★推奨
お店の活気をアピールするため、すぐには消さずに「売約済み」として表示を残します。
1. `cars.html` を開く。
2. 売れた車両の「車両詳細を見る」ボタン（`<a>`タグ）を、以下のような「SOLD OUT」のボタン（クリックできない状態）に書き換える。
   `<div class="flex-grow bg-gray-400 text-white py-2 rounded font-bold text-center">SOLD OUT（売約済み）</div>`
3. リンク（`href`）が外れるので、詳細ページへは飛べなくなります。
4. GitHubでセーブ（Commit & Push）する。

---

### 3. 車両一覧から完全に削除する手順（一定期間経過後など）
1. `cars.html` を開き、該当車両のHTMLコードのブロック全体を削除する。
2. 左側のファイル一覧から、該当する詳細ファイル（例：`car-detail-tanto-5678-202605.html`）を削除する。
3. GitHubでセーブ（Commit & Push）する。
